Longfield Solar Farm Examination library

An examination library lists all the documents (including comments, plans, evidence and statements) submitted about a nationally significant infrastructure project application. It covers the period from the point of application to the close of the examination phase.

Each document has an individual reference number. It's important to quote these reference numbers if you refer to any documents when making comments about a project.
