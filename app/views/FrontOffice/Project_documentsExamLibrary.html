Longfield Solar Farm Examination library

The examination library is a list of all the individual documents, including comments, plans, and statements.

Each item in the list is given an individual reference number. This is important because applications can include hundreds of individual comments, documents and evidence. People and organisations should quote any relevant examination library reference numbers in their comments so that everyone understands what documents the comment is referring to.
